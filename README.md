# Игра "Космические корабли"

## Суть игры
Два корабля передвигаются по экрану и стреляют друг в друга, пока у них не закончится "здоровье".


## Интерфейс
Игрок 1: управление стрелками, Ctrl - стрелять

Игрок 2: управление WASD, пробел - стрелять
